# API Reference 

## Base URL 
https://jsonplaceholder.typicode.com

## GET /users
Returns a list of users. 

**Status Codes** 
- 200 OK: Returns a list of users
- 404 Not Found: Endpoint does not exist 

### Request 
GET /users

### Response 
```json
[
  {
    "id": 1,
    "name": "Leanne Graham"
  }
]
```
